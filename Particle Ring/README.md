# Particle Ring Unity Project

A custom particle system in a ring shape.

You can adjust the following parameters to control the system:

- Direction: specify the direction in which particles  rotate
- Count: how many particles
- Size: how large is the particles
- Radius: the radius of the ring
- Spread: the width of the ring
